## 1. 系统概述

### 1.1 系统简介

IT项目成本估算监控系统是一个面向企业科技部门的B端Web应用系统，主要功能包括：

- **实施成本预估**：上传需求文档，AI智能解析功能点，自动计算项目工作量和成本
- **成本消耗预估**：OCR识别合同信息，预估项目成本消耗和燃尽情况
- **成本偏差监控**：监控项目实际成本与预算的偏差，AI提供优化建议
- **项目管理**：管理项目列表、成员、成本信息

### 1.2 核心特性

- 🤖 **AI驱动**：集成大语言模型，智能解析需求文档、生成工作项描述、提供偏差分析建议
- 📊 **可视化报表**：丰富的图表展示，支持Excel报告导出
- 🔐 **用户权限管理**：支持用户登录、权限控制、操作日志
- 📱 **响应式设计**：适配不同屏幕尺寸

## 4. 功能模块详解

### 4.1 用户认证模块

**路径**: `/login`  
**后端路由**: `/api/auth`

**功能**:
- 用户登录验证
- JWT Token生成
- 登录失败次数限制（5次失败锁定30分钟）
- 用户信息获取
- 密码修改

**默认账号**:
- 用户名: `admin`
- 密码: `admin123`

### 4.2 实施成本预估模块

**路径**: `/cost-estimate/*`  
**后端路由**: `/api/estimate`

#### 4.2.1 文件上传 (Upload)

**路径**: `/cost-estimate/upload`

**功能**:
- 上传需求文档（支持.doc/.docx格式）
- 创建项目和文档记录

#### 4.2.2 文档解析 (Parse)

**路径**: `/cost-estimate/parse-result`

**功能**:
- AI解析需求文档
- 识别项目名称、系统名称
- 提取功能模块和复杂度
- 支持手动调整模块信息

#### 4.2.3 参数配置 (Config)

**路径**: `/cost-estimate/config`

**功能**:
- 配置复杂度系数（简单/中等/复杂）
- 配置系统关联系数
- 配置流程系数
- 配置技术栈系数
- 配置单价（各角色日单价）
- 配置管理系数
- 支持保存为模板

#### 4.2.4 结果展示 (Result)

**路径**: `/cost-estimate/result`

**功能**:
- 展示总工作量、总成本、人月
- 团队工作量分布饼图
- 各阶段工作量柱状图
- 阶段详细分解表
- 合规校验
- 导出Excel报告（AI生成工作项描述）

### 4.3 成本消耗预估模块

**路径**: `/cost-consumption/*`  
**后端路由**: `/api/consumption`

#### 4.3.1 信息输入 (Input)

**路径**: `/cost-consumption/input`

**功能**:
- OCR识别合同图片
- 录入项目成本信息
- 配置团队成员

#### 4.3.2 结果展示 (Result)

**路径**: `/cost-consumption/result`

**功能**:
- 展示成本消耗预估
- 燃尽日期预测
- 团队成本分配

### 4.4 成本偏差监控模块

**路径**: `/cost-deviation/*`  
**后端路由**: `/api/deviation`

#### 4.4.1 信息输入 (Input)

**路径**: `/cost-deviation/input`

**功能**:
- 上传项目进度图片
- AI识别进度数据
- 配置基准线

#### 4.4.2 结果展示 (Result)

**路径**: `/cost-deviation/result`

**功能**:
- 展示偏差分析结果
- AI优化建议
- 偏差趋势图

### 4.5 项目管理模块

**路径**: `/project/*`  
**后端路由**: `/api/projects`

#### 4.5.1 项目列表 (List)

**路径**: `/project/list`

**功能**:
- 展示所有项目
- 筛选和搜索
- 查看项目详情

#### 4.5.2 项目详情 (Detail)

**路径**: `/project/detail/:projectId`

**功能**:
- 展示项目完整信息
- 成本预估结果
- 成本消耗情况
- 偏差监控数据

### 4.6 仪表盘模块

**路径**: `/dashboard`  
**后端路由**: `/api/dashboard`

**功能**:
- 项目统计概览
- 最近项目列表
- 待办事项提醒

### 4.7 个人设置模块

**路径**: `/user/setting`

**功能**:
- 修改个人信息
- 修改密码

## 10. 常见问题处理

### 10.1 后端服务无法启动

**检查项**:
1. 端口是否被占用：`lsof -i :3000`
2. 数据库文件是否存在：检查 `dev.db`
3. 环境变量是否配置正确：检查 `.env`
4. 依赖是否安装完整：`npm install`

**解决方案**:
```bash
# 杀掉占用端口的进程
kill -9 <PID>

# 重新初始化数据库
npm run db:push

# 检查日志
pm2 logs backend
```

### 10.2 前端页面空白

**检查项**:
1. 前端是否构建成功
2. Nginx配置是否正确
3. API代理是否正常

**解决方案**:
```bash
# 重新构建前端
cd SourceCode/frontend
npm run build

# 检查Nginx配置
sudo nginx -t

# 检查浏览器控制台错误
```

### 10.3 AI服务超时

**现象**: 导出Excel或其他AI操作超时

**原因**: AI模型推理耗时较长（约60秒）

**解决方案**:
1. 确保前端API超时设置为180秒
2. 确保Nginx代理超时设置 `proxy_read_timeout 180s`
3. 提示用户耐心等待

### 10.4 文件上传失败

**检查项**:
1. 上传目录权限
2. 文件大小限制
3. 文件格式是否支持

**解决方案**:
```bash
# 检查上传目录权限
chmod 755 SourceCode/backend/uploads

# 检查Nginx配置的文件大小限制
client_max_body_size 50M;
```

### 10.5 登录失败

**现象**: 提示"用户名或密码错误"或"账号已锁定"

**解决方案**:
1. 检查默认账号：admin / admin123
2. 账号锁定后等待30分钟或重置数据库
3. 检查密码加密是否正常

```bash
# 重置用户数据（开发环境）
sqlite3 dev.db "DELETE FROM users WHERE username='admin';"
npm run db:push  # 重新初始化
```
